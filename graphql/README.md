# Python Basic

## 1. Setup

### 1.1. Setup notebook

- Create Jupyter lab environment

  ```bash
  $ bash setup.sh
  ```

- Entry into the virtualenv

  ```bash
  $ source .venv/bin/activate
  ```

- Start jupyter lab

  ```bash
  $ jupyter lab --no-browser
  ```
